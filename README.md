# medic
Manage your Windows apps from command line. Useful for automation or if you afraid GUI.

## Usage
#### Get app list
```bash
medic list
```

#### Reset medic (remove cache, auth and all settings)
```bash
medic reset
```

## Install
```bash
sudo npm install medic -g
```

## Build
```bash
npm install
gulp # dev
gulp min # production
```
